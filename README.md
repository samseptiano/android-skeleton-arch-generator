# Android Skeleton Architecture Generator (Android Studio Plugin)

See demo video:
🎥 [Here](https://videy.co/v?id=fi3GrOms1)

Generate an Architecture Skeleton instantly inside Android Studio.

This plugin automatically creates the basic layers needed for a scalable Android architecture, so you no longer need to write the same boilerplate code every time you start a new feature or module.

# ✨ What It Generates

The plugin automatically generates the following architecture layers:

| File Generated |
| ------ |
|Model|
|Repository (Interface)|
|Repository Implementation|
|Use Case|
|View Model|
|Network Layer|

This helps maintain a clean, organized, and maintainable project structure.

# Compatibility
```sh
Android Studio Otter Feature Drop Ver. 2025.2.1 and above.
```
